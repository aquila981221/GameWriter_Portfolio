# 二、世界观设定

【故事背景】

- 明线：北宋末年，金军攻破汴京，俘虏徽、钦二帝，自此中原血红，天下大乱

- 暗线：原本应该重登紫府的一百单八星君，由于种种原因，并未功成果满，去邪归正，反倒因为心结未解，意气难平，堕入魔道，为祸一方。

【世界核心矛盾】

- 金兵肆虐，涂炭生灵
- 百姓被魔君们散播的业障侵蚀了心智，沦为受欲望支配的人形野兽，流毒四方

【主要势力】

- 抗金义军：以宗泽、岳飞为首，在黄河两岸组织义军，孤城坚守，抵御金人南侵，志在收复故土，重整山河
- 金军：意在消灭黄河以北的抵抗势力，为之后的南侵扫清障碍
- 世界的抑制力：以九天玄女为首，赐予玩家扮演的不死人“无名”无字天书，要他击败入魔的梁山好汉，将他们收入天书，拯救天下苍生
- 众魔君：以堕入魔道的梁山好汉为首，他们散播的业障如瘟疫般肆虐，不断侵蚀凡世。

【与游戏机制的结合】

玩家扮演的角色本是一具无名尸，被九天玄女救活，重塑肉身，所以才能一次次死而复生
