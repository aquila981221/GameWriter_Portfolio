# 六、任务脚本示例

【任务名称】郓哥的请托

【任务类型】支线任务·第N章

【任务相关地点】阳谷县-后巷、阳谷县郊-乱葬岗、阳谷县-紫石街-武家老屋

【前置条件】无

【任务主要奖励】武家老屋钥匙、武松遗书、拨浪鼓

【任务流程】

**Stage 1：巷中低语（两种触发路径）**

路径A：正常流程（先触发对话）

【触发条件】

玩家探索至一处阴暗的后巷，经过一扇紧闭的窗户时，被窗中的剪影叫住，玩家上前，触发剧情对话。

【对话脚本】

- 窗中剪影（郓哥）： “嘿，外乡人，过来……你也是来寻找那位“打虎英雄’的吗？”
- 无名（玩家）：“（默认交互）你知道他的下落？”

- 窗中剪影（郓哥）： “那当然，我们可是老相识了。他往西北郊的乱葬岗去了，你只管去那里寻他。”

路径B：捷径流程（先消灭怨灵）

【触发条件】

玩家未与窗中剪影（郓哥）对话，先在乱葬岗击败了由潘金莲、西门庆、王婆组成的恐怖怨灵。

【后续】玩家经过后巷，触发与窗中剪影（郓哥）对话时，将直接跳至“Stage 3 - 路径B”。

**Stage 2：乱葬岗的梦魇（针对路径A）**

【触发条件】玩家前往阳谷县西北郊外乱葬岗

【战斗设定】玩家遭遇难以名状的恐怖怨灵。怨灵长着无数手脚，拥有三个相互纠缠的头颅（妖娆妇人、浮浪官人、鸡皮鹤发的老婆子）。玩家将其击败。

**Stage 3：真相**

路径A：正常流程

【触发条件】

解决乱葬岗的恐怖怨灵，返回阳谷县后巷，再次与窗中剪影对话

【对话脚本】

- 窗中剪影（郓哥）： “你……你回来了？先前去了那么多人，你是第一个活着回来的！”
- 无名（玩家）：“武松到底在哪儿？”
- 窗中剪影（郓哥）： “实不相瞒，我也不晓得武都头去了哪儿，不过，我可以将武家在紫石街赁房的钥匙送给你。想当年二郎为兄报仇、刺配孟州牢城后，那鬼气森森的屋子就再也没人敢住。直到后来武松回来，又搬进去住了一段时日，不久前再度离开，临行前将这钥匙交给我保管。你若有胆，就自己去看看，说不定会有线索。”

【结果】玩家获得【武家老屋钥匙】

【尾声】玩家转身离开时，窗后传来的喃喃自语：

- 窗中剪影（郓哥）：“你这三寸丁，谷树皮，死了也不安生。如今那对奸夫淫妇，还有那个马泊六，皆已魂飞魄散，不会再合起伙来欺负你了，你也高抬贵手，休要再来纠缠！”

路径B：捷径流程

【触发条件】

先解决乱葬岗的恐怖怨灵，后探索至一处阴暗的后巷，经过一扇紧闭的窗户时，被窗中的剪影叫住，玩家上前，触发剧情对话。

【对话脚本】

- 窗中剪影（郓哥）： “嘿，外乡人，过来……（嗅了嗅，诧异）嘶，这股令人作呕的腐烂味，莫非……那乱葬岗的三个魔障已经被你干掉了？”
- 无名（玩家）：“（默认交互）是。”
- 窗中剪影（郓哥）： “太好了，总算（长出一口气）……你是在找打虎英雄武二郎吧？虽然我不知道他的下落，但我可以将武家在紫石街赁房的钥匙送给你。想当年二郎为兄报仇、刺配孟州牢城后，那鬼气森森的屋子就再也没人敢住。直到后来武松回来，又搬进去住了一段时日，不久前再度离开，临行前，将钥匙交给我保管。你若有胆，就自己去看看，说不定会有线索。”

【结果】玩家获得【武家老屋钥匙】

【尾声】玩家转身离开时，窗后传来的喃喃自语：

- 窗中剪影（郓哥）：“你这三寸丁，谷树皮，死了也不安生。如今那对奸夫淫妇，还有那个马泊六，皆已魂飞魄散，不会再合起伙来欺负你了，你也高抬贵手，休要再来纠缠！”

**Stage 4：武家老屋**

【触发条件】

玩家来到阳谷县-紫石街，用【武家老屋钥匙】打开武家老屋。

【调查】 发现武大的灵床子前有一张席子和许多空酒坛，暗示武松曾在此酗酒留宿。

【结果】 在席子旁拾取剧情道具【拨浪鼓】与【武松遗书】。

【任务完结】 

该支线补充了主线剧情，同时通过道具【拨浪鼓】与本章节最终BOSS战产生联动。在后续与武松的战斗中，若玩家在二阶段摇响拨浪鼓，武松将短暂恢复神智，产生硬直，为玩家提供宝贵的输出机会。
