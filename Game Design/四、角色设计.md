# 四、角色设计

【名称】“行者”武松

【所在地】景阳冈-乱树林

【相关地图】阳谷县、景阳冈

【场景氛围】

狂风在山林间穿梭，仿佛鬼啸，中央是一片空地，有块光滑的大青石，边上是那头被无名打死的恶虎，小山般隆起的身躯软趴趴地堆成一团

【出场方式】

恶虎已死，无名正要走，听到动静回头，只见一只大手从虎尸体内破膛而出，在喷涌的血雾中，猛地撕开虎腹，钻出一个威武雄壮、浑身浴血的行者。他手执梢棒，仰面朝天，喉咙里发出一声深沉的叹息，缓缓睁开野兽般的眼眸。

【视觉关键词】

- 整体轮廓：人形，身高两米，虎背熊腰

- 感觉：威武、阴沉、凶狠、压迫感

- 物品材质：梢棒（原木色）、顶骨数珠（白色）、戒箍（黑铁）、两把戒刀（烂银）、乌云皂直裰（黑色）、斑斓布衲袄（多种颜色错杂）

- 细节：一身虎血，袒露胸膛

- 特效：周身缭绕着不详的黑雾

【角色描述】

身躯凛凛，相貌堂堂。一双眼光射寒星，两弯眉浑如刷漆。胸脯横阔，有万夫难敌之威风；语话轩昂，吐千丈凌云之志气。心雄胆大，似撼天狮子下云端；骨健筋强，如摇地貔貅临座上。如同天上降魔主，真是人间太岁神。

【背景故事】

武松，一百单八将之一，绰号行者。清河县人士，家中行二，又叫“武二郎”。曾在景阳冈上赤手空拳打死一头吊睛白额大虫，又于阳谷县怒杀潘金莲、西门庆，为兄长武大报仇。后刺配孟州牢城，血溅鸳鸯楼，为躲避官府追捕，扮作头陀。梁山大聚义时，排第十四位，上应天伤星，司职步军头领。梁山散伙后，武松四处漂泊，因魔心未断，又造下许多杀孽。感到罪孽深重的他回到阳谷县，决心舍身饲虎，以赎罪孽，不曾想，却惹出更大的祸端。

【战斗设计】

- 一阶段：武松拖着梢棒，闲庭信步，云淡风轻，如老僧禅定，以静制动。攻击欲望不高，多为试探性进攻，但偶尔会使出密不透风的多段连招，让玩家猝不及防，瞬间感受到巨大压力。血线达到50%，进入CG动画，武松奋力一击，梢棒折断

- 二阶段：武松进入癫狂状态，攻击方式改为拳脚，韧性与进攻欲望极大增强，多为高速突进、长距离跳跃和抓投技，还有跳砸、战争践踏等大范围攻击，玩家的输出窗口被压缩。若完成【郓哥的请托】支线任务，可使用道具【拨浪鼓】，使武松短暂恢复神智。二阶段血线达到0，进入CG动画，武松恢复神智，重回巅峰，拔出腰间两口雪花镔铁戒刀

- 三阶段：武松的刀法大开大阖，刚猛无俦，所有攻击命中玩家后，将叠加流血效果。可在地面或空中挥出中远距离的血红剑气，多段连招和大范围攻击应接不暇，中间冷不丁一个高速突进或长距离跳跃，逼至玩家跟前，接以抓取和投技，令人防不胜防。若玩家血量低于三分之一，且被武松抓投命中，有概率触发处决技，一刀斩下玩家首级。 

【语音台词设计】

- 战斗中：

  却饶你不得！

  纳命来！

  一不做，二不休，杀了一百个，也不差你这一个！

- 击杀玩家：杀人者，打虎武松也！

- 死亡：武松，去也……

【相关物品】

【雪花镔铁戒刀】

- 物品描述：武松专属武器，由镔铁打造的两口戒刀，烂银也似刀刃布满雪花般的纹理。此刀杀人无数，时至今日，依然会在半夜里发出不详的鸣啸。

- 专属战技：挥出两道月牙形的血红剑气，形成交叉十字，再次发动能使出更多连击。普通攻击与战技命中后都会给敌人叠加流血效果

- 获取方式：阳谷县-景阳冈-乱树林-击败武松

【行者套】

- 物品描述：直裰冷披黑雾，戒箍光射秋霜。额前剪发拂眉长，脑后护头齐项。顶骨数珠灿白，杂绒绦结微黄。钢刀两口迸寒光，行者武松形象。

- 特殊效果：装备之后，稍微提升暴击率和流血效果

- 获取方式：阳谷县-景阳冈-乱树林-击败武松

【拨浪鼓】

- 物品描述：武家老屋遗落的拨浪鼓。武氏兄弟自幼父母双亡，武大含辛茹苦将武二抚养长大。儿时武松哭闹，一听哥哥摇响拨浪鼓，便会平静下来。斯人已逝，孑然一身的行者如今只能睹物思人。

- 特殊效果：使入魔的武松短暂恢复神智

- 获取方式：阳谷县-紫石街-武家老屋-完成【郓哥的请托】支线任务
