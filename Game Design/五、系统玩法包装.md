# 五、系统玩法包装

【系统一：无字天书/角色升级/BOSS再挑战】

- 系统名称：无字天书

- 系统描述：九天玄女授予无名之天书，用于收服入魔的三十六天罡、七十二地煞，并可从中汲取力量，磨练自身

- 角色升级功能：玩家每击败一名梁山好汉，将其收入无字天书之中，便可获得其一部分能力，具体表现为一定数量的属性点，用于提升属性

- BOSS再挑战：已经击败并收入天书的梁山好汉，玩家可以进入天书再次挑战

【系统二：好汉遗物/特殊装备/解锁结局】

- 系统名称：好汉遗物

- 系统描述：击败梁山好汉后，获得其掉落之遗物。将好汉们的这些武器、甲胄带回梁山，摆放在忠义堂里，昔日大聚义的盛况犹在眼前

- 特殊装备功能：与普通装备不同，玩家装备好汉遗物，可触发特殊效果

- 解锁结局：将梁山好汉的遗物全部放回忠义堂，使其尽数归位，方可解锁最终结局

【系统三：装备强化】

- 系统名称：装备强化

- 系统描述：昔日梁山，有那金钱豹子汤隆做铁匠总管，提督打造诸般军器，并铁叶连环等甲

- 装备强化功能：救得汤隆，入驻水泊梁山之后，便可去他处强化装备
